### MultiPhotoFrame ###

===========================================================================
DESCRIPTION:

This project demonstrates the use of the multi-image dragging API introduced in Lion. It shows how to change the dragging image for your destination view and animate the dragging image from the drop location to its final location in the view while simultaneously animating a gap for the dropped items. Also learn how to start a multi-image drag and change the image when the drag leaves the view.

===========================================================================
BUILD REQUIREMENTS:

Mac OS X 10.8 or later

===========================================================================
RUNTIME REQUIREMENTS:

Mac OS X 10.7 or later

PACKAGING LIST:

MulitPhotoView.h/m
